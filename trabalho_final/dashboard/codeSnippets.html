Code availability

In order to improve the reproducibility of this work, all codes and data used are available on the Internet. The query to search the OS base posts can be accessed at this link https://data.stackexchange.com/stackoverflow/query/1124873/ehealth-discussions. The codes used to build the visualizations presented in the article are available on Observable Notebook through the observablehq.com/@pedroalmir and the datasets can be found at: https://github.com/pedroalmir/datavis-course.
